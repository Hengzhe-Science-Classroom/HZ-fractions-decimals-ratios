# Fractions, Decimals & Ratios · Interactive

**Visual fraction models, decimal operations, and proportional reasoning**

**Status:** Planned

## What You'll Learn

- Fraction models: area, number line, set models
- Decimal place value and operations
- Ratio and proportion
- Percent, rate, and unit pricing
- Cross-multiplication and scaling
